# 软件分析技术 2018 课程大作业

* 作业说明 [软件分析技术（熊英飞）](https://xiongyingfei.github.io/SA/2018/main.htm)
* 请用 IntelliJ 打开本工程
* 请将 `src/main/java` 目录标为源代码目录
* 请将 `code` 目录标为测试代码目录
* 运行时请运行 `src/main/java/RunPointerAnalysis.java`
* `code` 目录下存放的是样例测试数据

## 命令行参数

* 第一个参数指定代码所在目录
* 第二个参数指定 `main()` 方法所在的类
* 示例 `out/test/pointerAnalysis test.Hello`
